# 🎯 Ticket Resolution Time Prediction

## 📌 Problem Statement
Predict the time required to resolve customer support tickets using machine learning.

## 📊 Dataset
Customer support ticket dataset containing:
- Ticket Priority
- Ticket Type
- Response Time
- Customer Details

## ⚙️ Approach
- Data Cleaning  
- Feature Engineering  
- Encoding categorical variables  
- Model Building (Linear Regression, Random Forest)  
- Model Evaluation (MAE, MSE)  

## 🤖 Models Used
- Linear Regression  
- Random Forest Regressor  

## 📈 Results
- Linear Regression MAE: ~8.0  
- Random Forest MAE: ~7.85 (Better Performance)  

## 💡 Business Impact
This model helps predict resolution time in advance,
allowing better SLA management and resource planning.
